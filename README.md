# CareerCompass

CareerCompass: AI-Powered Career Advisor is an interactive chatbot designed to provide
personalised career guidance. Using a Retrieval-Augmented Generation (RAG) system, it
integrates an open-source Large Language Model (LLM) with structured data from the
O*NET 29.1 database. Users can ask career-related questions, explore job roles,
understand required skills, and receive tailored recommendations based on industry trends.
The system will be designed to be scalable, open-source, and accessible to students, job
seekers, and professionals looking for career insights. Additionally, users can view an
interactive dashboard to learn more about job industries and careers through key figures
and visualizations.

## Project Goals

1. A Fully Functional Career Guidance Chatbot
2. A Structured and easy to query Career Database
3. A User-Friendly Frontend
4. A Dashboard with Visualizations
5. Documentation and Deployment Guide
